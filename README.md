<h1 align="center">T-20-Cricket-Performance-Analysis</h1>
<h3 align="center">A Player Analysis Project.</h3>



## Project Overview
Cricket is one of the most data-rich sports in the world — yet most performance decisions are still made on intuition. This project dives into T20 international cricket data sourced from ESPN Cricinfo to analytically evaluate player performance across key batting and bowling metrics.
The goal was not just to describe the data, but to extract actionable insights — identifying standout performers, consistency patterns, and what statistically separates
match-winners from average players.


## Tools & Technologies
- Power BI  (Interactive dashboard & visual storytelling)
- Jupyter Notebook  (Interactive EDA — step-by-step analysis with inline visualizations) 
- Python/Pandas  (Data Cleaning and Transformation)


## Steps involved in the project
1) Requirement Scoping

2) Data Collection using Web Scraping from ESPN Cricinfo website
   
3) Data Cleaning and Preprocessing in Pandas
   
4) Data Transformation in Power Query
   
5) Data Modelling and Building Parameters in Power BI using DAX
   
6) Building the Dashboard in Power BI
